## Instalação
1. Clone o Projeto em um diretorio de sua escolha.
2. Abra um terminal no VSCode para cada um dos [microserviços](# Microserviços)
3. Em cada terminal vincule a pasta do seu respectivo microserviço com:
```
 cd <pasta-do-microsserviço>
```
4. Instale as dependência de cada um dos microserviços com:
```
 npm install
```
5. Inicie os microserviços em cada terminal com:
```
 npm start
```
6. Agora é só fazer as requisições utilizando o aplicativo de sua preferência (recomendamos o ThunderClient).
